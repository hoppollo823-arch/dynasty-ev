# 班公湖场景技术规格书 — DynastyVR POC

> **项目：** DynastyVR  
> **场景：** 班公湖（Pangong Tso）  
> **引擎：** Unreal Engine 5.x  
> **版本：** v1.0 — 2026-03-18  
> **状态：** 🔴 POC 首场景，最高优先级

## 1. 场景地理参数

| 参数 | 数值 |
|---|---|
| GPS 坐标 | 33.7°N, 78.9°E |
| 湖面海拔 | 4,242 m |
| 湖面面积 | ~604 km² |
| 最大深度 | ~33 m |
| 东西长度 | ~150 km |
| 南北宽度 | ~3–5 km |
| 湖水颜色成因 | 盐度梯度（东淡西咸） |

**盐度梯度与颜色关系：**

班公湖是典型的内流湖，东部有河流淡水注入，盐度低、颜色浅蓝；向西逐渐盐化，颜色加深至深蓝甚至碧绿。这一色彩渐变是班公湖最具辨识度的视觉特征，必须在场景中忠实还原。

## 2. 地形建模

### 2.1 湖盆形态

- **整体形态：** 狭长型，东西向延伸约 150 km
- **南北宽度：** 约 3–5 km
- **湖岸特征：**
  - **北岸：** 较平缓，碎石滩地貌，可步行
  - **南岸：** 较陡峭，岩石直接切入水中
- **远景山脉：** 冈底斯山脉余脉，海拔 5,000–6,000 m

### 2.2 核心精修区域

为控制 POC 开发范围，选定 **3 km² 核心区域**：

| 属性 | 值 |
|---|---|
| 中心坐标 | 33.75°N, 78.95°E（东端大桥附近） |
| 覆盖范围 | 桥梁 + 两侧湖岸各 ~800m |
| 精修内容 | 全精度地形、植被、人造物、交互点 |

### 2.3 地形数据源

- **基础地形：** SRTM 30m DEM 或 ALOS 12.5m DEM
- **高程修正：** 参考 Google Earth 高程数据
- **湖底地形：** 模拟碗形，中心最深 33m，边缘渐浅
- **地形分层：**
  - Layer 0: 湖底（沉积物/盐晶纹理）
  - Layer 1: 湖岸（碎石/沙地）
  - Layer 2: 山坡（岩石/荒漠）
  - Layer 3: 雪山（积雪/冰川）

## 3. 水体系统（UE5 Water System）

### 3.1 湖水颜色渐变

使用 **Material Parameter Collection (MPC)** 控制沿湖长度的颜色渐变。

| 湖段 | 位置 | 颜色值 (Linear) | Hex 近似 |
|---|---|---|---|
| 东段 | 大桥附近 | R:0.4 G:0.7 B:0.9 | #66B2E6 |
| 中段 | 湖中部 | R:0.2 G:0.6 B:0.8 | #3399CC |
| 西段 | 湖西端 | R:0.1 G:0.4 B:0.7 | #1A66B3 |

**实现方式：**
- 基于 World Position X 轴偏移量驱动颜色插值
- MPC `BangongLake_WaterColor` 参数实时控制，支持日夜切换
- 日落时分颜色向金色/橙色偏移（R+0.2, G+0.1, B-0.1）

### 3.2 水面波纹

| 参数 | 值 |
|---|---|
| 波浪模型 | Gerstner Wave |
| 主风向 | 西北（315°） |
| 波浪幅度 | 0.1–0.3 m（平静），0.5–0.8 m（有风） |
| 波长 | 2–5 m |
| 波浪叠加层数 | 3 层（主波 + 细波 + 微波） |

### 3.3 光学属性

| 属性 | 值 |
|---|---|
| 透明度 | 清澈，水下可见深度 2–3 m |
| 反射率 | 高（F0 ≈ 0.02），完美倒影雪山 |
| 折射率 | 1.333（标准水体） |
| 次表面散射 | 低（清澈水体，非浑浊） |
| 焦散效果 | 强（高海拔日照直射） |

### 3.4 水下内容

- **简化处理**，非交互区域
- 底部纹理：盐结晶 + 碎石
- 使用半透明体积雾模拟水下能见度衰减
- 粒子系统：微小悬浮颗粒（盐晶闪光）

## 4. 山脉与积雪

### 4.1 近处山脉

| 属性 | 值 |
|---|---|
| 主色调 | 褐色 / 土黄色（Hex: #8B7355） |
| 植被覆盖 | 几乎无植被（<5%） |
| 岩石类型 | 花岗岩/片麻岩质感 |
| 法线强度 | 高（粗糙岩面） |
| 位移贴图 | 细节级 0.3–0.5m 凸起 |

**材质层次：**
1. Base Color: 褐色岩石纹理（4K tiling）
2. Normal Map: 高频岩石细节
3. Displacement: 中频地貌起伏
4. Roughness: 0.7–0.9（粗糙表面）

### 4.2 雪山与积雪

| 属性 | 值 |
|---|---|
| 雪线海拔 | ~5,500 m |
| 雪线以下 | 裸露岩石 |
| 雪线以上 | 永久积雪 / 冰川 |
| 积雪材质 | Mossy/Rocky 混合，程序化分布 |
| 分布算法 | 基于海拔 + 坡度 + 面向（阴面雪更多） |
| 积雪粗糙度 | 0.2–0.4（光滑雪面） |
| 积雪高光 | 强（阳光下刺眼反射） |

**程序化积雪参数：**
```
SnowAltitude = 5500.0
SnowSlopeMax = 45.0          // 坡度 > 45° 积不住雪
SnowNormalInfluence = 0.3     // 法线朝向影响
SnowNoiseScale = 100.0        // 噪声边界不规则度
SnowBlendWidth = 200.0        // 雪线过渡带（米）
```

## 5. 天空与大气

### 5.1 高海拔大气

使用 UE5 **Sky Atmosphere** 系统，模拟海拔 4,200m 的稀薄大气。

| 参数 | 调整方向 |
|---|---|
| Rayleigh Scattering | 减少密度（稀薄大气散射弱） |
| 天顶颜色 | 深蓝（比低海拔更暗的蓝色） |
| 地平线颜色 | 偏白（尘埃散射） |
| 太阳强度 | 增强 1.15x（紫外线强） |
| 臭氧吸收 | 微调增强短波吸收 |

### 5.2 云层

| 类型 | 频率 | 特征 |
|---|---|---|
| 卷云（Cirrus） | 常驻 | 高空、稀疏、丝状，白色半透明 |
| 积雨云 | 偶发 | 远处山脉上方，10% 概率 |
| 无云区 | 主体 | 大部分天空晴朗 |

### 5.3 光照系统

| 时段 | 太阳角度 | 色温 | 湖面效果 |
|---|---|---|---|
| 正午 | 高角度 | 5,800K（冷白） | 强反射、刺眼 |
| 日落 | 低角度 | 3,200K（暖金） | 湖面变金色/橙红 |
| 黄昏 | 地平线下 | 2,800K | 天空渐变紫红 |
| 夜晚 | 月光 | 4,100K | 湖面微光反射 |

**日落效果要点：**
- 太阳接近地平线时，湖水颜色从蓝渐变为金橙
- 山脉轮廓出现剪影效果
- 雪山被染成粉红/金色（Alpenglow 效果）

### 5.4 后处理体积

- **Exposure:** 自动曝光，范围 1.0–8.0（高海拔强光适应）
- **Bloom:** 强（日照直射 + 雪面反射）
- **Volumetric Fog:** 低密度，模拟远处大气透视
- **Vignette:** 轻微

## 6. 植被与生物

### 6.1 植被

班公湖周边植被极其稀少，符合高原荒漠特征。

| 类型 | 密度 | 位置 |
|---|---|---|
| 高山灌木 | 极低（<10株/100m²） | 北岸碎石缝中 |
| 草甸斑块 | 低（<5%覆盖） | 湖岸平缓地带 |
| 苔藓 | 微量 | 岩石背阴面 |

**资产建议：**
- Megascans 高山灌木包
- Nanite 草甸变体（LOD0）
- Instanced Static Mesh（性能优化）

### 6.2 动物

| 动物 | 数量 | 行为 | 实现方式 |
|---|---|---|---|
| 海鸥 | 5–10只 | 空中盘旋、偶尔降落水面 | AI 控制 + 骨骼动画 |
| 野驴（藏野驴）| 2–3头 | 远处缓步移动 | AI 巡逻路径 |

- 海鸥使用 **行为树**：盘旋 → 俯冲 → 起飞循环
- 野驴使用 **Spline 路径**，沿湖岸缓步
- 距离 > 200m 时切换为低精度动画

### 6.3 经幡

在关键位置放置藏传佛教五色经幡：

| 位置 | 数量 | 朝向 |
|---|---|---|
| 大桥入口 | 2 组 | 桥两侧 |
| 湖岸高地 | 1 组 | 面湖 |
| 营地附近 | 1 组 | 面东 |

- 经幡使用 **Cloth Simulation**（风向西北驱动）
- 五色顺序：蓝、白、红、绿、黄（天空→火焰→大地）
- 颜色做旧处理（风吹日晒褪色感）

## 8. LOD 策略与性能预算

### 8.1 LOD 层级

| 层级 | 距离 | 三角面/对象 | 说明 |
|---|---|---|---|
| **L0** | 玩家位置 | 全精度 | Nanite 启用，4K 纹理 |
| **L1** | < 100 m | 中等精度 | LOD1 mesh，2K 纹理 |
| **L2** | < 1 km | 简化 | LOD2 mesh，1K 纹理 |
| **L3** | > 1 km | Billboard | Distance Field + Imposter |
| **L4** | > 5 km | 最简 | 平面 + 颜色块 |

### 8.2 具体资产 LOD 配置

| 资产 | L0 | L1 | L2 | L3+ |
|---|---|---|---|---|
| 湖面 | 全波纹+反射 | 简化波纹 | 平面色块 | Billboard |
| 核心地形 | Nanite 全精度 | LOD1 | LOD2 | Heightmap |
| 山脉 | Nanite | LOD1 | LOD2 | Billboard |
| 大桥 | 10K tri | 3K tri | 1K tri | Billboard |
| 海鸥 | 骨骼动画 | 简化骨骼 | 静态 | 无 |
| 经幡 | Cloth Sim | 无Cloth | 静态 | 无 |

### 8.3 性能预算

| 类别 | 预算 | 目标 |
|---|---|---|
| **场景内存** | < 2 GB | 此场景总计 |
| 地形 | < 800 MB | Nanite + Virtual Texture |
| 水体 | < 200 MB | Water Body + Foam |
| 山脉/雪山 | < 500 MB | Nanite 简化 |
| 人造物 | < 200 MB | L0 资产 |
| 植被 | < 100 MB | Instanced |
| 动物 | < 50 MB | 简化动画 |
| 大气/天空 | < 100 MB | Sky Atmosphere |
| 缓冲 | ~50 MB | — |

### 8.4 渲染性能目标

| 指标 | 目标值 |
|---|---|
| 帧率 | ≥ 60 FPS |
| Draw Calls | < 2,000 |
| 着色器复杂度 | Medium |
| 光照复杂度 | Medium（主光源 + 天光） |
| 阴影 | 级联阴影（近处），距离场阴影（远处） |
