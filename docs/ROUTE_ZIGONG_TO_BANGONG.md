# 🚗 自贡→班公湖 自驾路线

> 陛下钦定路线 · 奔驰迈巴赫 QES 680

## 车辆信息
- **车型：** Mercedes-Maybach S680 / EQS 680
- **动力：** V12 双涡轮 / 纯电长续航
- **颜色建议：** 双色车漆（曜岩黑/银砂金）
- **特殊准备：** 高海拔适配、备用轮胎、应急氧气

## 路线总览

```
自贡 → 乐山 → 大理 → 香格里拉 → 拉萨 → 阿里大北线 → 神山圣湖 → 狮泉河 → 班公湖
总里程：约 6,500+ km
建议天数：25-30天
```

## 详细路线

### 第一段：自贡 → 大理（约900km）

| 站点 | 里程 | 说明 |
|------|------|------|
| 🏮 **自贡彩灯公园** | 起点 | 出发，夜间彩灯闪耀 |
| **乐山** | ~150km | 乐山大佛远眺，岷江汇入 |
| G5 京昆高速 | ~600km | 经西昌、攀枝花 |
| **大理古城** | ~900km | 抵达，苍山洱海 |

**沿途景色：**
- 岷江河谷田园风光
- 攀西大裂谷干热河谷
- 滇西高原草甸过渡

### 第八段：神山 → 狮泉河 → 班公湖（约500km）

| 站点 | 说明 |
|------|------|

**这段景色：**

- 班公湖：东西延伸，中印各占一半

## UE5 游戏内实现要点

1. **路网数据导入** — G318/G219/G5 精确建模
2. **沿途地形** — 大北线段需手工精修（现有卫星数据不足）
3. **天气系统** — 高原极端天气（暴风雪/沙尘/强紫外线）
4. **车辆损伤系统** — 路面碎石/涉水/高海拔动力衰减
5. **加油站/补给点** — 真实位置还原
6. **野生动物群** — 藏野驴/藏羚羊/秃鹫程序化生成
7. **星空** — 大北线段银河/极光效果
8. **迈巴赫细节** — 内饰交互/车载导航/空调制暖
