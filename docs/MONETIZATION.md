## 一、商业模式总览

### 核心模式：免费游玩 + 虚拟商品 + 订阅制

DynastyVR 采用 **F2P（Free-to-Play）+ 虚拟经济 + VIP 订阅** 三层变现架构，确保低门槛进入、高天花板付费。

```
┌─────────────────────────────────────────────────────────┐
│                   DynastyVR 商业模型                      │
│                                                          │
│   ┌──────────┐   ┌──────────┐   ┌──────────┐            │
│   │ 免费层   │   │ 虚拟商品 │   │ VIP订阅  │            │
│   │          │   │          │   │          │            │
│   │ 探索地球 │   │ 地块交易 │   │ 月卡/年卡│            │
│   │ 基础飞行 │   │ 车辆商城 │   │ 专属特权 │            │
│   │ 公共区域 │   │ UGC商城  │   │ 去广告   │            │
│   │ 基础社交 │   │ 活动门票 │   │ 加速解锁 │            │
│   └──────────┘   └──────────┘   └──────────┘            │
│        ↑              ↑              ↑                  │
│   ─── 拉新 ───→ ─── 付费转化 ───→ ─── 留存/ARPU ───→   │
└─────────────────────────────────────────────────────────┘
```

### 收入来源矩阵

| 收入类型 | 占比（稳态） | 特点 |
|----------|:-----------:|------|

## 二、虚拟货币系统：帝国币 (Dynasty Coin)

### 2.1 货币设计

帝国币（Dynasty Coin / DC）是 DynastyVR 内部经济体系的核心通货，支撑所有虚拟商品和服务的流通。

| 属性 | 说明 |
|------|------|
| **名称** | 帝国币 / Dynasty Coin / DC |
| **符号** | 🪙（UI 显示为 DC 图标） |
| **最小单位** | 1 DC = 1 币 |
| **充值比例** | 1 CNY ≈ 10 DC（基准汇率） |
| **汇率浮动** | 固定充值汇率，拍卖市场自由浮动 |
| **总供应量** | 无上限（法币锚定，非限量通缩模型） |
| **有效期** | 永久不过期 |

### 2.2 获取途径

#### A. 法币充值（主要收入来源）

| 充值档位 | 金额 (CNY) | 获得 DC | 赠送 DC | 折扣率 |
|:--------:|:---------:|:-------:|:-------:|:------:|
| 首充 | 6 | 60 | +60（首充翻倍） | 50% |
| 小额 | 30 | 300 | +30 | 9% |
| 中额 | 68 | 680 | +100 | 13% |
| 大额 | 198 | 1,980 | +400 | 17% |
| 豪华 | 648 | 6,480 | +2,000 | 24% |
| 帝王充 | 3,288 | 32,880 | +12,000 | 27% |

#### B. 游戏内获取（免费途径）

| 途径 | DC 奖励 | 频率 |
|------|:-------:|------|
| 每日登录 | 10-50 DC | 每日递增（7日循环） |
| 完成探索任务 | 20-200 DC | 任务制 |
| 自驾里程奖励 | 每 100km = 5 DC | 无上限 |
| 飞行任务 | 50-500 DC | 挑战制 |
| 成就解锁 | 100-5,000 DC | 一次性 |
| 社交邀请 | 100 DC/人 | 每月上限 1,000 |
| UGC 作品点赞 | 按热度分池 | 周结 |
| 观看广告 | 5-20 DC/次 | 每日上限 10 次 |

#### C. 活动产出

- 节日活动限定 DC 红包
- 排行榜赛季奖励
- 官方赛事奖金池

### 2.3 消费场景

```
帝国币消费流向
├── 🏗️ 地块购买/竞拍
├── 🚗 车辆购买/租赁/改装
├── 🎨 车辆皮肤/贴纸
├── 🏠 建筑材料/家具
├── 👤 Avatar 外观/动作
├── 🎫 活动门票
├── 🎬 UGC 作品购买
├── 📦 开箱/抽奖（受监管）
├── 🔄 跨平台资产同步手续费
└── ⭐ VIP 订阅（月卡/年卡）
```

### 2.4 经济平衡机制

| 机制 | 说明 |
|------|------|
| **通胀控制** | DC 只能通过充值和游戏内产出，不能无限刷取；每日产出设上限 |
| **回收机制** | 地块维护费、车辆保养费、VIP 订阅等持续消耗 DC |
| **汇率锚定** | 充值汇率固定，拍卖市场价格自由波动 |
| **反作弊** | 异常产出检测、交易冷却期、大额交易人工审核 |
| **储备金池** | 平台持有 DC 储备用于市场调控 |

## 三、地块交易系统

### 3.1 虚拟地产定义

DynastyVR 中的数字地球每一个坐标点都可划分为可交易地块，覆盖全球主要城市和景点。

| 地块类型 | 面积范围 | 示例位置 |
|----------|:--------:|----------|
| **黄金地块** | 500-2,000 m² | 曼哈顿时代广场、上海外滩、东京涩谷 |
| **核心地块** | 1,000-5,000 m² | 陆家嘴金融区、巴黎香榭丽舍、伦敦金丝雀码头 |
| **标准地块** | 2,000-10,000 m² | 城市住宅区、商业区 |
| **郊外地块** | 5,000-50,000 m² | 郊区、乡村、风景区 |
| **景区地块** | 1,000-10,000 m² | 班公湖畔、那拉提草原、阿里神山 |

### 3.2 地价体系

#### 影响因子权重

| 因子 | 权重 | 说明 |
|------|:----:|------|
| 地理位置稀缺性 | 30% | 全球知名地标 > 一般城市 > 郊区 |
| 周边设施 | 20% | 靠近热门景点/交通枢纽加分 |
| 地块面积 | 15% | 大地块单价低但总价高 |
| 交通可达性 | 15% | 是否临近主干道/机场 |
| 人气指数 | 10% | 历史访问量、关注度 |
| 时间衰减 | 10% | 每季度微调，反映市场热度 |

#### 地价等级（DC 参考价）

| 等级 | 区域 | 起拍价 (DC) | 预估成交价 (DC) |
|:----:|------|:-----------:|:---------------:|
| S 级 | 曼哈顿时代广场 | 5,000,000 | 20,000,000 - 50,000,000 |
| S 级 | 上海外滩/陆家嘴 | 4,000,000 | 15,000,000 - 40,000,000 |
| A 级 | 东京银座/巴黎香街 | 2,000,000 | 8,000,000 - 20,000,000 |
| A 级 | 迪拜哈利法塔 | 1,500,000 | 5,000,000 - 15,000,000 |
| B 级 | 城市核心住宅区 | 200,000 | 800,000 - 3,000,000 |
| C 级 | 郊外/景区 | 50,000 | 100,000 - 500,000 |

### 3.3 拍卖系统

#### 拍卖类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| **英式拍卖** | 公开竞价，价高者得，60秒倒计时 | 热门地块首次出售 |
| **荷式拍卖** | 起拍高价逐降，首个应价者得 | 批量地块、冷门区域 |
| **密封拍卖** | 出价不可见，截止时最高者得 | 特殊/限量地块 |
| **直接购买** | 固定价格，即买即得 | 一般地块、市场挂牌 |

#### 拍卖流程

```
卖家挂拍 → 平台审核 → 设定底价/时间 → 竞拍开始
                                          ↓
                               公开展示（3-7天）
                                          ↓
                              竞拍结束 → 最高价成交
                                          ↓
                            买家支付 DC → 产权转移
                                          ↓
                            平台收取 5% 手续费
```

### 3.4 产权规则

| 规则 | 说明 |
|------|------|
| 产权期限 | 购买后永久拥有（服务器运营期间） |
| 维护费用 | 每月收取基础维护费（面积 × 单价 × 0.1%） |
| 空置惩罚 | 超过 30 天未登录，维护费翻倍 |
| 转让交易 | 支持玩家间自由交易，平台收取 5% 手续费 |
| 继承机制 | 支持指定继承人或平台回收 |
| 回购机制 | 平台以购入价 50% 回购（限购入 30 天内） |

## 八、广告植入系统

### 8.1 设计原则

> **非侵入式广告** — 广告即场景的一部分，是数字世界的真实倒影。

DynastyVR 中的广告遵循以下原则：

| 原则 | 说明 |
|------|------|
| **场景融入** | 广告以虚拟广告牌、品牌门店等形式自然存在于城市中 |
| **用户可控** | VIP 用户可关闭广告；所有用户可调整广告偏好 |
