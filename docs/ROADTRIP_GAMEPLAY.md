## 1. 驾驶手感设计

### 1.1 迈巴赫 QES 680 物理参数

| 参数 | 数值 | 备注 |
|------|------|------|
| 整备质量 | ~2,700 kg | 豪车级别，惯性大 |
| 发动机 | V12 双涡轮增压 6.0L | 或纯电版本 4WD |
| 最大功率 | ~612 PS (V12) / ~500 kW (纯电) | 高原衰减见 1.3 |
| 最大扭矩 | ~900 Nm | 起步推背感强烈 |
| 悬挂 | AIRMATIC 空气悬挂 + E-Active Body Control | 可主动调节软硬 |
| 轴距 | ~3,400 mm | 转弯半径大，弯道需预判 |
| 驱动 | 4MATIC 全时四驱 | 越野路段有优势 |
| 油箱 | ~80 L (V12) | 纯电版电池 ~120 kWh |

#### UE5 物理实现

- **质量惯性**：车辆变道/转弯时应感受到 2.7 吨的重量转移
- **悬挂行程**：空气悬挂高程 4 级可调（舒适/标准/运动/越野），越野模式抬升 30mm
- **轮胎模型**：不同路面摩擦系数差异明显（见 1.2）
- **制动力分配**：前后 60:40 比例，长下坡时后刹温度上升更快

### 1.2 路面反馈差异

#### 柏油路（高速 / G318 部分路段）

- 车辆最稳定，方向盘轻盈精准
- AIRMATIC 舒适模式，过滤细碎震动
- 时速 120+ 时方向盘略微加重，增加稳定感
- 噪音极低，车内可清晰听到音响

#### 砂石路（大北线 / 乡道）

- 方向盘抖动加剧，需要持续微调
- 碎石击打底盘的声音反馈（随机石子飞溅音效）
- 悬挂频繁压缩回弹，视觉上车身晃动
- **轮胎损伤累积**：碎石路面持续行驶降低轮胎耐久度
- 越野模式自动启用，离地间隙增加
- 空气悬挂切换至"越野"硬设定

#### 涉水路段（大北线/山区溪流穿越）

- 进入水面前有浅滩标识，玩家需判断水深
- 水面阻力使车速骤降
- **水深指示器**：仪表盘显示当前水深 vs 车辆最大涉水深度（约 400mm）
- 水花溅起效果覆盖引擎盖
- 涉水后车辆变脏（泥点飞溅）
- 持续涉水可能导致发动机进水 → 触发故障

### 1.3 高海拔动力衰减

**核心机制：功率随海拔线性衰减**

```
有效功率 = 基准功率 × (1 - 0.008 × 海拔(km))

示例：
- 0m (自贡)：612 PS × 1.00 = 612 PS
- 3,300m (香格里拉)：612 × 0.974 = 596 PS
- 4,700m (措勤)：612 × 0.962 = 589 PS → 实际感受为 ~12% 衰减
- 5,200m (大北线垭口)：612 × 0.958 = 586 PS
```

**游戏表现：**
- 仪表盘显示实时功率限制百分比
- 爬坡时加速明显迟缓，玩家需降挡或提前加速
- 涡轮增压车型（V12）衰减较涡轮介入前明显，涡轮介入后部分补偿
- 纯电版本：高海拔对电机功率影响小，但电池低温衰减更严重

### 1.4 温度影响

#### 高寒启动（-20°C 以下，大北线冬季/高海拔清晨）

- 启动时间延长 3-5 秒
- 仪表盘显示"预热中"进度条
- 冷车状态怠速偏高（~1200 RPM → 缓慢降至 ~800 RPM）
- 油耗增加 ~15%
- 转向助力略重

#### 空调制热耗能

- 低温环境下空调自动制热启动
- 纯电版本：制热消耗显著续航（每小时约 -5~8% 电量）
- V12 版本：利用发动机余热，耗能较小但怠速增加
- **策略要素**：玩家需在保暖和能耗间取舍
- 大北线夜间扎营时：可选择怠速供暖（耗油）或裹睡袋（不耗油但降低角色舒适度）

## 2. 车内交互

### 2.1 数字仪表盘

仪表盘采用迈巴赫风格全液晶显示，可自定义布局：

**默认信息：**
- **时速** — 中央大数字，MPH/KMH 可切换
- **转速** — 弧形条，红线区 6,500 RPM
- **油量/电量** — 右侧条形图，<15% 变红闪烁
- **海拔** — 实时高度计（对垭口翻越至关重要）
- **指南针** — 方位角 + 前方地名标注
- **挡位** — P/R/N/D/M + 手动模式显示

**可选信息：**
- 涡轮压力 / 电机输出功率
- 悬挂高度等级
- 轮胎压力四轮独立显示
- 发动机/电池温度
- 当前油耗/电耗（L/100km 或 kWh/100km）
- 预计续航里程

### 2.2 中控导航屏幕

- **路线指引**：显示全程路线，可缩放/拖拽
- **卫星地图模式**：真实卫星影像贴图（大北线段使用高程数据）
- **兴趣点标注**：加油站、观景台、城镇
- **ETA 计算**：基于当前速度和剩余路线实时更新
- **海拔剖面图**：显示前方路线的海拔变化（预判垭口难度）
- **天气预警**：前方路段天气信息推送

### 2.3 空调系统

- 温度/风量/出风口方向可调
- **高海拔自动制热**：检测到海拔 >4,000m 且气温 <5°C 时自动开启暖风
- 后排独立空调（乘客模式下可自行调节）
- 空气净化系统（显示 PM2.5 数值）
- **能耗提示**：纯电版空调高功率运行时显示续航影响

### 2.4 音响系统

- **音乐播放**：可导入自定义播放列表
- **FM 收音机**：模拟搜索，不同地区有不同电台（进入藏区后信号渐弱）
- **音量随速调节**：高速自动增大抵消风噪
- **Burmester 3D 环绕声**：可切换环绕/立体声模式
- 大北线无信号区：收音机变为白噪声，增添孤独氛围

### 2.5 后视镜/倒车影像

- **流媒体后视镜**：后方广角实时画面
- **倒车影像**：挂 R 档自动激活，带辅助线和距离提示
- **360° 环视**：鸟瞰视角查看车辆周围
- **越野模式**：侧方摄像头显示车轮位置，辅助过窄路

### 2.6 天窗控制（看星星）🌟

- 全景天窗，可一键开启 / 分段开启
- **夜间观星模式**：停车 + 天窗全开时触发
  - 大北线段：银河效果逼真（根据真实星图渲染）
  - 可旋转视角观赏不同星座
  - 音响自动切换为环境音（风声+虫鸣）
- **延时关闭**：设定自动关闭时间（防止夜间忘记关）

### 2.7 座椅按摩

- **5 种按摩模式**：波浪/脉冲/定点/全背/腰椎
- **长时间驾驶自动提醒**：连续驾驶 >2h 后提示"建议开启座椅按摩缓解疲劳"
- **疲劳度系统**：按摩可降低角色疲劳值
- 大北线长途段（300km+ 无补给）按摩功能成为关键减压手段
- 后排座椅同样配备（乘客模式）

## 3. 车辆状态系统

### 3.1 油量/电量管理

#### 补给点规划

| 加油站 | 距上一站 | 建议 | 风险 |
|--------|----------|------|------|
| 🏮 **自贡**（起点） | — | ⛽ 满油出发 | — |
| **乐山** | ~150 km | ⛽ 补给 | 低风险，城市加油站密集 |
| 大理 | ~750 km | ⛽ 补给 | 高速沿途有保障 |
| 香格里拉 | ~350 km | ⛽ 必须加满 | 进入藏区前最后城市级加油站 |
| **拉萨** | ~1,600 km | ⛽⛽ **必须满油** | 大北线前最后补给！ |
| 措勤 | ~600 km | ⛽ 小镇加油站 | 油质一般，价格偏高 |
| 亚热 | ~400 km | ⛽ 最后补给 | 可能仅有柴油 |
| 狮泉河 | ~500 km | ⛽ 阿里首府 | 补给充足 |

#### 油量管理机制

- **剩余续航提示**：仪表盘始终显示"可行驶里程"
- **低油量警告**：
  - <30%：黄色提示"建议尽快加油"
  - <15%：红色闪烁 + 语音提示
  - <5%：发动机限功率（最高 80 km/h）
- **油耗动态计算**：
  - 城市道路：~15 L/100km
  - 高速巡航：~11 L/100km
  - 大北线砂石路：~18-22 L/100km（路面阻力大）
  - 爬坡路段：~25 L/100km
  - 怠速供暖：~2 L/h

#### 大北线 300km 无加油站路段

这是全行程最关键的燃油管理挑战：

- 进入该路段前**必须确认油量 >60%**
- 仪表盘弹出路线警告，显示最近加油站距离
- 若油量不足 → 玩家可：
  1. 返回措勤/亚热补油
  2. 路边遇到牧民/越野车队时求援（随机事件）
  3. 严重缺油时触发救援任务（游戏内等待/小游戏）

### 3.2 轮胎状态

- **耐久度**：100% → 0%，持续显示
- **影响因素**：
  - 柏油路：耐久度几乎不损耗
  - 砂石路：持续缓慢损耗
  - 重度碎石路（大北线）：加速损耗
  - 涉水后路面：损耗加速
- **轮胎压力**：
  - 正常：2.5 bar
  - 高海拔降压：海拔每升 1,000m，胎压自动降低 ~0.1 bar（物理模拟）
  - 爆胎风险：<1.8 bar 时风险增大
- **修复**：
  - 携带补胎工具可修复轻微损伤
  - 严重损坏需前往城镇修理（大北线段可能需等 1-2 天游戏时间）

### 3.3 发动机/电机温度

- **正常范围**：90-105°C
- **高海拔散热问题**：
  - 空气稀薄 → 散热效率下降
  - 连续爬坡时温度快速上升
  - 超过 115°C：仪表盘红色警告
  - 超过 125°C：触发限功率保护
  - **应对策略**：停车怠速冷却 / 降低转速行驶
- **纯电版**：电池温度管理
  - 低温（< -10°C）：电池效率下降 20-30%
  - 高负载：电池温度上升，需监控

### 3.4 车辆清洁度

- **清洁度等级**：Clean → Dusty → Dirty → Muddy → Filthy
- **影响因素**：
  - 柏油路：不脏
  - 砂石路：缓慢变脏
  - 涉水后：显著变脏
  - 雨天/泥泞路段：快速变脏
- **视觉表现**：
  - 车身逐渐出现灰尘/泥点
  - 车窗变脏 → 降低能见度
  - 前大灯被泥覆盖 → 照明效果减弱
- **清洗**：
  - 城镇加油站有洗车服务
  - 大北线段可能无洗车 → 脏车成为常态，增加真实感
  - 河边手动洗车（小游戏 / 耗时但不花钱）

## 4. 驾驶挑战设计

### 4.1 垭口翻越 🏔️

**场景特征**：陡坡 + 连续弯道 + 低气压

**机制设计：**
- 坡度指示器：显示当前坡度百分比（10%、15%、20%...）
- 上坡时：
  - 动力受限（高海拔衰减叠加爬坡负荷）
  - 需要提前加速冲坡
  - 速度过低时提示"降挡"
- 下坡时：
  - 刹车温度监控（连续下坡 → 刹车过热 → 制动力衰减）
  - 提示"使用低挡发动机制动"
  - 刹车温度 >600°C 时发出警告
- 弯道：
  - 视野受限（山体遮挡）
  - 对向来车可能出现（卡车/越野车）
  - 迈巴赫转弯半径大 → 窄路掉头需技巧
- **气压变化**：海拔升高，车内气压缓慢变化 → 耳鸣音效

**关键垭口节点：**

| 垭口 | 海拔 | 难度 | 特殊要素 |
|------|------|------|----------|
| 色季拉山口 | 4,728m | ★★★ | 远眺南迦巴瓦峰 |
| 东达山 | 5,130m | ★★★★ | G318 最高点 |
| 大北线各垭口 | 5,000-5,200m | ★★★★★ | 无护栏、搓板路 |

### 4.2 涉水路段 💧

**水深指示系统：**
- 仪表盘实时显示水深（通过声纳/摄像头）
- 车辆最大涉水深度：400mm（空气悬挂越野模式）
- 颜色编码：
  - 🟢 <200mm：安全
  - 🟡 200-350mm：需谨慎
  - 🔴 >350mm：危险

**操作要点：**
- 入水前减速，保持匀速通过
- 过快 → 水花溅入进气口 → 发动机故障
- 过慢 → 车辆可能被水流推动
- 通过后刹车片需烘干（连续轻踩刹车动画）

### 4.3 大北线搓板路 🌊

**搓板路（Washboard road）物理模拟：**
- 路面呈周期性波纹
- **速度敏感**：
  - 低速（20-30 km/h）：颠簸明显但可控
  - 中速（40-60 km/h）：共振区间，车身剧烈弹跳
  - 高速（>70 km/h）：可能短暂"漂浮"，失控风险极高
- **最佳策略**：找到共振速度窗口之外的舒适速度
- **悬挂反馈**：空气悬挂持续工作，可听到压缩/回弹声
- **车内物品**：中控杯架的水杯晃动、后视镜抖动
- **车辆损耗**：持续行驶加速零件疲劳

### 4.4 夜间驾驶 🌙

**光照系统：**
- 迈巴赫数字大灯（Digital Light）：
  - 近光：覆盖前方 60m，铺路均匀
  - 远光：覆盖前方 200m，照射宽度增加
  - **自动远近光**：检测到对向车辆时自动切近光
- 仪表盘/中控自动调暗
- 路面反光效果（潮湿路面 / 冰面反射）

**路边动物出没：**
- 大北线段夜间：
  - 🦌 藏羚羊群横穿公路（随机）
  - 🐺 狼群（远处绿眼反光）
  - 🦅 秃鹫栖息路边（静态点缀）
- **反应机制**：
  - 急刹车 / 猛打方向 → 可能翻车
  - 正确操作：减速 + 等待动物通过
  - 撞击动物：车辆受损 + 道德扣分

### 4.5 恶劣天气 🌪️

#### 暴风雪

- 能见度骤降至 50m 以内
- 路面结冰 → 摩擦系数降至 0.15
- ABS/ESP 频繁介入
- 风挡积雪 → 需开启雨刮 + 前窗加热
- **应对**：减速/停车等待 / 寻找避风处

#### 沙尘暴（大北线/阿里）

- 能见度极低（20-30m）
- 黄橙色调笼罩画面
- 风阻增大 → 车身偏移
- 空气滤芯负担增加（模拟指标）
- **应对**：开启双闪、降低车速、依赖导航 GPS 行驶

#### 强紫外线（高原晴天）

- 不影响驾驶但增加氛围真实感
- 风挡反光 → 需调整遮阳板
- 长时间暴晒 → 车内温度升高 → 空调负载增加

## 5. 自动驾驶 / 乘客模式 🤖

### 5.1 自动驾驶模式

**触发条件：**
- 高速公路或路况良好的铺装路段
- 玩家手动切换或语音指令"开启自动驾驶"

**功能：**
- 车辆按当前路线自动巡航
- 自动保持车距 / 车道居中
- 自动变速 / 适应限速
- 遇到障碍物自动减速/避让

**玩家视角自由：**
- 切换至**自由视角**：可 360° 旋转观察窗外风景
- 第一/第三人称视角自由切换
- 可放大查看路边地标/动物
- 可查看手机地图 / 听音乐 / 与乘客互动
- **画中画**：小窗显示前方路况（保持安全感）

### 5.2 乘客模式

**触发条件：**
- 车辆静止时切换"乘客席"
- 或指定 AI 驾驶员接管

**乘客可操作：**
- 🪟 车窗升降（感受风/温度变化）
- 🌟 天窗开关（观星/通风）
- 🎵 音响控制（选歌/调音量）
- 🌡️ 空调调节（自己的出风口）
- 💺 座椅按摩
- 📸 拍照功能（截图 + 滤镜）

**乘客视角特有体验：**
- 观看后排娱乐屏幕
- 透过车窗看风景 + 自动慢动作捕捉精彩瞬间
- 与 AI 乘客对话（如有同行角色）

## 6. 途经点互动 📍

### 6.1 打卡拍照系统

**自动触发：**
- 到达预设打卡点时：
  - 仪表盘弹出"📍 已到达 [地点名]"
  - 屏幕右上角提示"按 [P] 拍照"
- **拍照模式**：
  - 冻结当前画面
  - 自由调整相机角度/焦距
  - 内置滤镜（日落/黑白/胶片/高原蓝）

## 7. 情感体验设计 🎭

### 7.1 第一天的兴奋（自贡 → 大理，~900km）

**情绪基调：期待、兴奋、自由**

- **出发仪式**：
  - 自贡彩灯公园夜景中启动车辆
  - 仪表盘显示"旅程开始"动画
  - 音乐自动播放欢快曲目
- **沿途感受**：
  - 高速巡航的畅快感
  - 窗外风景从四川盆地逐渐变为滇西高原
  - 温度逐渐变化（盆地闷热 → 高原凉爽）
- **到达大理**：
  - 苍山洱海全景展现
  - 提示"第一天完成 ✅"
  - 奖励解锁：大理古城 360° 全景相框

### 7.2 进入藏区的敬畏（香格里拉 → 拉萨，~1,600km）

**情绪基调：敬畏、好奇、逐渐适应**

- **香格里拉过渡**：
  - 景色从绿意盎然变为高原草甸
  - 经幡开始出现
  - 藏式建筑替代汉族风格
- **海拔攀升**：
  - 仪表盘海拔数字不断上升
  - 耳鸣音效 + 画面轻微呼吸感
  - 角色出现轻微高反（画面边缘微红）
- **壮丽景观冲击**：
  - 南迦巴瓦峰 / 雅鲁藏布江
  - 巴松措碧绿湖水
  - 布达拉宫远眺
- **环境音变化**：
  - 城市噪音消失 → 虫鸣/风声/水声
  - 隐藏 BGM 变为低沉的藏族颂钵

### 7.3 大北线的孤独感（无人区，~1,400km）

**情绪基调：孤独、紧张、原始**

- **进入无人区**：
  - 手机信号彻底消失
  - 收音机只剩白噪声
  - 仪表盘提示"前方 300km 无加油站"
  - 画面色调变冷/荒凉
- **感官设计**：
  - **静**：极端安静，只有风声和轮胎碾压碎石声
  - **空**：窗外无尽荒原，偶尔出现一个碧蓝湖泊
  - **远**：地平线极远，天与地几乎融为一体
- **孤独反馈**：
  - 长时间无人对话 → 角色自言自语（可关闭）
  - 夜晚扎营时只有车灯照亮一小片区域
  - 看到其他车辆时产生强烈安全感
- **紧张元素**：
  - 车辆故障时的无助感
  - 油量焦虑
  - 野生动物突然出现

### 7.4 到达神山的震撼（冈仁波齐）

**情绪基调：神圣、渺小、感动**

- **冈仁波齐出现**：
  - 经过漫长荒原后，金字塔形山体逐渐显现
  - **音乐渐起**：藏族宗教音乐 + 管弦乐叠加
  - 画面雾气散开，阳光照射山体
  - 时间可能放慢（慢镜头感）
- **玛旁雍错**：
  - 圣湖碧蓝水面 + 冈仁波齐倒影
  - 环境音变为诵经声（远处）
- **情感触发**：
  - 角色发出感叹语音
  - 画面自动提供最佳拍照角度
  - 解锁"朝圣者"成就

### 7.5 终点的成就感（班公湖）🏆

**情绪基调：释然、自豪、不舍**

- **接近终点**：
  - 狮泉河补给后最后一段
  - 湖水蓝色逐渐出现
  - 里程碑提示"距班公湖 50km... 30km... 10km..."
- **到达班公湖**：
  - 湖水渐变蓝 + 雪山环绕全景展现
  - **音乐高潮**：旅程 BGM 终章版本
  - 仪表盘弹出动画：
    - "🏁 到达班公湖"
    - "总里程：6,xxx km"
    - "最高海拔：5,200m"
    - "旅程天数：xx 天"
- **结算画面**：
  - 旅程回放（关键节点自动剪辑）
  - 收集品统计（照片/打卡点/里程数）
  - 成就列表
  - 生成旅程海报（可分享）
- **彩蛋**：
  - 站在班公湖边远眺对岸
  - 角色语音："从自贡到这里... 6500 公里... 值了。"
  - 画面渐黑，字幕滚动
