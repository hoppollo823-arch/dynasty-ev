## 核心支柱

### 🌍 一、真实地球 (Real Earth)

**目标：闭眼无法分辨数字还是真实**

#### 地形系统
- 数据源：Copernicus DEM (30m) + 本地LiDAR精化
- 城市区域：Photogrammetry扫描（倾斜摄影）
- 自然区域：AI地形生成 + 卫星影像贴图
- 海洋：实时海浪模拟 + 海底地形

#### 建筑系统
- 全球主要城市：OpenStreetMap + Google 3D Tiles
- 精细区域（纽约时代广场、上海外滩）：手工精修AAA模型
- 住宅区：AI根据真实卫星图生成建筑

#### 植被系统
- 全球植被分布：基于MODIS卫星数据
- 区域精化：AI生成符合气候带的真实植被
- 动态季节：根据经纬度+时间自动切换

#### 大气系统
- 物理大气散射（瑞利+米氏散射）
- 实时云层（卫星云图驱动）
- 日夜交替、黄金时刻、蓝调时刻
- 城市灯光（夜晚自动点亮）

#### 交通系统
- 真实公路网络导入（OpenStreetMap）
- 高速公路、国道、乡道全部还原
- 桥梁、隧道、收费站精确建模
- 实时车流（AI填充）

### 🚗 二、自驾体验 (Road Trip)

**从自贡出发，走318国道到班公湖，再去伊利结婚。**

#### 路线系统
- 真实路网：基于OpenStreetMap + 高德/百度路网数据
- 可规划路线：输入起点终点，自动规划
- 途经点：自由添加中途停靠

#### 车辆系统
- 豪车车库：法拉利、兰博基尼、奔驰、路虎...
- 真实物理：悬挂、轮胎、四驱系统
- 驾驶模式：方向盘/手柄/VR手柄操控
- 车内视角：精致内饰，可切换乘客视角

#### 沿途风景（重点）
| 路段 | 景色还原 |
|------|----------|
| 自贡市区 | 四合院、盐业建筑、夜景灯光 |
| 雅安→康定 | 二郎山隧道、大渡河峡谷 |
| 折多山 | 雪山垭口、经幡、牦牛 |
| 理塘→稻城 | 高原草甸、白塔、藏式村落 |
| 班公湖 | 湖水渐变蓝、雪山倒影、鸥鸟 |
| 伊利草原 | 薰衣草田、那拉提草原、蒙古包 |

#### 天气联动
- 实时天气API接入（或模拟历史天气）
- 高海拔区域：暴风雪、路面结冰
- 真实日照角度

### ✈️ 三、飞行体验 (Flight)

**从纽约起飞，穿越大西洋，飞越喜马拉雅，降落在阿里。**

#### 飞行器
- 私人飞机：湾流G650、庞巴迪环球
- 直升机：AW139、西科斯基
- 太空船：可选，用于地球→太空过渡
- 座舱：完全交互，仪表盘可用

#### 飞行路线示例：纽约→阿里
```
纽约肯尼迪机场起飞
  → 大西洋（看日出从海平面升起）
  → 冰岛（看火山、冰川）
  → 北欧（看峡湾）
  → 俄罗斯冻土带
  → 贝加尔湖（冰面或碧蓝湖水）
  → 蒙古草原
  → 昆仑山脉（雪山连绵）
  → 喜马拉雅（8000m级山峰从窗外掠过）
  → 冈仁波齐（神山）
  → 阿里高原（降落）
```

#### 大气效果
- 随高度变化的大气密度
- 平流层：看到地球弧度
- 云层穿梭：积雨云、卷云分层
- 光照：不同高度阳光折射

### 🕳️ 四、宇宙探索 (Cosmos)

**飞出大气层，前往黑洞。**

#### 太空模块
- 近地轨道：看地球全景、极光
- 月球：真实月面地形（LRO数据）
- 火星：全彩高精地形
- 土星环：粒子系统渲染

#### 黑洞可视化
- 数据基础：EHT（事件视界望远镜）真实影像
- 渲染效果：
  - 引力透镜（背景星光弯曲）
  - 吸积盘（高温等离子体发光）
  - 光子环（光子被捕获的轨迹）
  - 霍金辐射（微弱粒子喷流）
- 交互：接近→时间变慢、潮汐力拉伸视觉效果
- 可选：Schwarzschild 黑洞 vs Kerr 黑洞（旋转黑洞）

#### 星空
- 基于真实星表（Hipparcos/Tycho-2）
- 任意位置看星空 = 真实该位置看到的星空
- 银河系全景
- 星云：猎户座、鹰状星云等（哈勃/JWST数据）

### 🏠 五、建造系统 (Build)

**在纽约、上海任何地方，造自己的豪宅。**

#### 地块系统
- 全球可购买地块（虚拟产权）
- 纽约曼哈顿、上海陆家嘴... 热门地段
- 地块大小可调

#### 建造工具
- 模块化建筑：拖拽式放置墙壁、地板、屋顶
- 自由建造：支持复杂几何体
- 室内设计：家具、灯光、材质
- 风格模板：现代简约、中式园林、欧式古典...

#### 豪车车库（豪宅配套）
- 停车在自家豪宅车库
- 驾车出门即是真实路网
- 车辆定制：颜色、内饰、改装

### 💍 六、人生事件 (Life Events)

**去伊利结婚。**

- 婚礼系统：选定地点、布置场地、邀请宾客
- 宾客入场：好友从全球各地飞来参加
- 仪式流程：宣誓、交换戒指、合影
- 纪念功能：婚礼录像、照片自动生成
- 扩展：生日派对、周年庆、毕业典礼...
