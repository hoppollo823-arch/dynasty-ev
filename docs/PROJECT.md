# 🌍 DynastyVR — 全球实景社交平台

> 东方皇朝 EV 元宇宙部核心项目

## 项目定位

构建一款 VR 社交软件，内置类 Google Earth VR 的全球实景漫游能力，用户可在虚拟地球中探索真实世界场景，并与好友实时社交互动。

## 核心功能模块

### 1. 🌐 全球实景引擎 (GeoEngine)
- **地球导航**：从太空缩放到街景的无缝过渡
- **3D 地形渲染**：基于 Cesium / Google Earth API 的真实地形
- **街景模式**：360° 全景街景沉浸体验
- **航飞模式**：自由飞行探索
- **搜索定位**：语音/文字搜索全球任意地点

### 2. 👥 社交系统 (Social)
- **多人在线**：好友同屏共游地球
- **语音聊天**：空间化 3D 语音（距离衰减）
- **Avatar 系统**：虚拟形象 + 手势追踪
- **表情系统**：面部追踪 + 手势表情
- **社交动作**：指向、挥手、合影等

### 3. 🏠 虚拟空间 (Spaces)
- **用户主页**：自定义虚拟家园
- **兴趣广场**：按主题聚类的社交空间
- **活动场地**：虚拟旅游团、线上聚会
- **UGC 场景**：用户可在真实地点上叠加虚拟内容

### 4. 📱 跨平台支持
- **VR 头显**：Meta Quest、PICO、Apple Vision Pro
- **PC VR**：SteamVR 兼容
- **移动端**：手机/平板 2D 模式（降级体验）
- **Web**：浏览器 WebXR 入口

## 技术选型

| 组件 | 选型 | 理由 |
|------|------|------|
| 游戏引擎 | Unity 2022 LTS + HDRP | VR 成熟生态，跨平台强 |
| 地球引擎 | Cesium for Unity | 开源 3D 地球，支持 3D Tiles |
| 网络框架 | Unity Netcode for GameObjects | 官方方案，与引擎深度集成 |
| 语音 | Vivox / Photon Voice | 空间化语音，VR 验证方案 |
| 后端 | Go + gRPC | 高性能，微服务友好 |
| 数据库 | PostgreSQL + PostGIS | 地理数据支持 |
| 缓存 | Redis | 在线状态、会话管理 |
| CDN | Cloudflare | 全球地图瓦片加速 |
