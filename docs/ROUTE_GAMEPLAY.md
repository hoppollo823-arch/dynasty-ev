## 一、核心玩法循环

```
驾驶 → 到达 → 探索 → 收集 → 解锁 → 继续前行
  ↑                                          │
  └──────────── 目的地到达 ←─────────────────┘
```

### 1.1 玩法循环拆解

| 阶段 | 玩家行为 | 系统反馈 |
|------|---------|---------|
| **驾驶** | 手动驾驶/辅助驾驶，操控迈巴赫在真实路网行驶 | 实时风景流式加载、天气变化、路面反馈 |
| **到达** | 抵达途经点/站点，触发到达事件 | 地标介绍动画、打卡记录、里程结算 |
| **探索** | 下车步行/进入建筑/与NPC互动 | 触发剧情、小游戏、隐藏收集 |
| **收集** | 拾取物品、拍照打卡、完成挑战 | 收集进度更新、成就解锁、背包系统 |
| **解锁** | 达成条件后解锁新区域/新成就/新明信片 | 里程奖励、路线图推进、社交分享提示 |
| **前行** | 选择继续前往下一站 | 新路段风景加载、驾驶事件开始 |

### 1.2 里程奖励系统

每行驶 **100km** 自动解锁一项奖励，奖励类型轮换：

| 里程碑 | 奖励类型 | 示例 |
|--------|---------|------|
| 100km | 风景明信片 | 「岷江河谷」明信片 |
| 200km | 成就徽章 | 「初出茅庐」铜徽章 |
| 300km | 地方特产 | 自贡井盐 |
| 400km | 风景明信片 | 「攀西大裂谷」明信片 |
| 500km | 成就徽章 | 「高原行者」银徽章 |
| ... | ... | ... |
| 6500km | 终极成就 | 「天路征服者」金徽章 + 专属车漆 |

**特殊里程：**
- **500km** — 车辆首次保养（教学事件）
- **1000km** — 解锁自定义路线分享功能
- **3000km** — 进入大北线前警告（油量/物资提醒）
- **6500km** — 终点解锁「自贡至班公湖·全程纪念册」

### 1.3 途经点打卡系统

- 每个途经点有 **「打卡」** 按钮，触发拍照视角 + 坐标记录
- 打卡后该点标记在路线图上，可回溯查看
- **连续打卡** 奖励：连续打卡5站解锁「打卡达人」成就
- **全打卡** 奖励：全程途经点100%打卡解锁「完美旅途」成就 + 隐藏车漆
- 打卡照片自动生成到 **「旅途相册」**，可导出分享

## 四、挑战系统

### 4.1 恶劣天气驾驶挑战

| 天气类型 | 触发路段 | 影响 | 奖励 |
|---------|---------|------|------|
| 暴风雪 | 折多山垭口、大北线垭口 | 能见度↓、路面结冰、车速限制 | 「暴风骑士」成就 |
| 沙尘暴 | 大北线荒原段 | 能见度极低、车辆积灰 | 「沙尘征服者」成就 |
| 强降雨 | 云南段 | 路面湿滑、山体滑坡风险 | — |
| 高原强紫外线 | 海拔4000m+全程 | 车内温度升高、需开启空调 | — |

### 4.2 垭口翻越（高海拔体力值管理）

| 机制 | 说明 |
|------|------|
| 体力值 | 海拔4000m以上缓慢下降，需休息/吸氧恢复 |
| 高反系统 | 体力值<30%时画面模糊、操作灵敏度下降 |
| 恢复方式 | 休息（停车等待）、吸氧瓶（消耗品）、下山恢复 |
| 关键垭口 | 折多山(4298m)、色季拉山(4728m)、马攸木拉(5211m) |

### 4.3 油量管理

| 机制 | 说明 |
|------|------|
| 适用路段 | 大北线段强制启用，其余路段可选 |
| 油耗计算 | 根据车速、坡度、载重实时计算 |
| 加油站 | 按真实位置还原，大北线最长间隔300km |
| 策略 | 需在每个加油站加满，精打细算 |
| 紧急事件 | 油量耗尽→呼叫救援（扣分）或步行至最近驿站 |

### 4.4 越野路段（可选难度）

| 难度 | 路段 | 要求 | 奖励 |
|------|------|------|------|
| 简单 | 全程主路线 | 无 | — |
| 普通 | 大北线部分非铺装路 | 四驱模式 | 大北线隐藏湖泊解锁 |
| 困难 | 当惹雍错支线 | 越野改装 | 象雄遗址深度探索 |
| 地狱 | 鬼湖环线 | 全改装+驾驶技术 | 隐藏龙形幻影彩蛋 |

### 4.5 夜间驾驶挑战

| 机制 | 说明 |
|------|------|
| 夜间模式 | 日落后自动启用，车灯照明范围有限 |
| 路况 | 大北线夜间无路灯，需全程开远光灯 |
| 挑战 | 夜间需注意野生动物横穿公路（紧急刹车事件） |
| 奖励 | 夜间行驶100km解锁「夜行者」成就 |

## 六、可玩性量化目标

### 6.1 交互密度

| 指标 | 目标值 | 设计保障 |
|------|--------|---------|
| 每小时可交互点 | ≥ 1个 | 每个主要站点至少3个交互点 |
| 每段路程无事件最长 | ≤ 20分钟 | 长路段插入驾驶事件（风景解说、天气变化、野生动物出没） |
| 全程主线游戏时长 | 25-30小时 | 不探索支线，仅到达+打卡+收集 |
| 完全探索时长 | 60-80小时 | 全收集+全支线+全成就+社交体验 |

| 乐山→大理 | 750km | 7h | 2h | 9h |
| 大理→香格里拉 | 350km | 4h | 3h | 7h |
| 香格里拉→拉萨 | 1600km | 16h | 5h | 21h |
